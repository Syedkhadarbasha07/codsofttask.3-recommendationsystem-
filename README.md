# 🎬 Movie Recommendation System

This project implements a simple Movie Recommendation System using Collaborative Filtering in Python.

## 🚀 Features
- User similarity calculation
- Movie recommendations based on ratings
- Collaborative filtering algorithm
- Command-line interaction

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## ▶️ How to Run

1. Clone repository:
git clone https://github.com/Syedkhadarbasha07/codsofttask.3-recommendationsystem-.git

2. Install dependencies:
pip install -r requirements.txt

3. Run program:
python recommendation.py

## 📊 Output
The system recommends movies based on similar users' preferences.
